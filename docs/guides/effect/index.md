# Effect-TS Guides

This directory contains guides and best practices for using Effect-TS in our codebase.

## Table of Contents

1. [Lifecycle Management](./001-lifecycle.md) - Managing application lifecycle, graceful shutdown, and signal handling

## Overview

Effect-TS is a powerful library for building robust TypeScript applications with built-in support for:
- Dependency injection
- Error handling
- Resource management
- Concurrent programming
- Testing

These guides document our learnings and established patterns for working with Effect in this project.
